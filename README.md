# 🛍️ UMKM Toko Baju — Web App by Rizki Ramadhan Zafitra

**UMKM Toko Baju** adalah sebuah web aplikasi sederhana yang ditujukan untuk membantu pelaku usaha mikro, kecil, dan menengah (UMKM) dalam mengelola toko pakaian secara digital. Dibangun menggunakan **PHP**, **MySQL**, dan **HTML/CSS/JavaScript**, aplikasi ini menyediakan fitur manajemen produk pakaian wanita dari sisi admin.

---

## 🔧 Fitur Utama

### 👩🏻‍💼 Admin Panel
- **Tambah Produk Wanita**  
  Admin dapat menambahkan produk seperti kaos, blouse, dress, dan lainnya.
- **Edit Produk**  
  Update nama, harga, atau gambar produk jika terjadi perubahan stok atau harga.
- **Hapus Produk**  
  Menghapus produk yang sudah tidak dijual.
- **Laporan Produk** *(Report)*  
  Menampilkan daftar produk yang sudah ditambahkan dalam bentuk tabel untuk keperluan pencatatan atau evaluasi.

## 🔐 Login Admin
Fitur login sudah diterapkan menggunakan form sederhana dan pengecekan database. Setelah berhasil login, admin diarahkan ke `dashboard.php`.

---

## 🚀 Cara Menjalankan
1. Jalankan WAMP/XAMPP.
2. Salin folder `toko_baju` ke direktori `www` atau `htdocs`.
3. Import file `db.sql` ke phpMyAdmin.
4. Akses `http://localhost/toko_baju/login.php`.
5. Login sebagai admin dan mulai kelola produk.
