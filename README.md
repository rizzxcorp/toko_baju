**UMKM Toko Baju** adalah aplikasi web sederhana untuk membantu pelaku UMKM menjual produk baju secara online. Dibuat dengan **PHP** dan **MySQL**, aplikasi ini memiliki fitur CRUD produk untuk admin.

## ✨ Fitur
- 🔐 Login Admin
- ➕ Tambah Produk Wanita
- 📝 Edit Produk (Nama, Harga, Gambar)
- ❌ Hapus Produk
- 📄 Laporan Produk

## 🛠️ Teknologi
- PHP + MySQL (Backend)
- HTML, CSS, JS (Frontend)
- WAMP/XAMPP (Local Server)
- Tailwind CSS (Planned for styling)

## 📂 Struktur Folder
```
📁 toko_baju/
├── index.php
├── dashboard.php
├── login.php
├── proses_tambah.php
├── update_harga.php
├── hapus_produk.php
├── db.php
├── db.sql
├── assets/
│   ├── images/
│   └── icons/
```

## 🚀 Cara Menjalankan
1. Jalankan WAMP/XAMPP
2. Letakkan folder di `www/` atau `htdocs/`
3. Import `db.sql` via phpMyAdmin
4. Akses `http://localhost/toko_baju/login.php`
